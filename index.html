A bit about me

A Web Developer in training, and a former flooring installer by trade.

Over 10 years ago I was asked to make a Website for my father, for whom I was apprenticing.

Almost immediately I was bitten by the bug, and for the next ten years, I slowly taught myself newer and better methods for completing the task of building a website.

Using localhost servers like xampp, I used online resources to learn bits and pieces of everything from MySql and PHP, to Javascript, CSS, Jquery, and HTML5.

I'm currently working to improve and perfect those skills in a proper learning environment, via the University of Toronto's SCS Coding Bootcamp.

This course has been a valuable resource for filling in the gaps of what I already knew, as well as exposing me to many new and exciting concepts.

Already we've covered things like :

  • Responsive design
  • Github and GitLab
  • Advanced CSS
  • Semantic HTML
